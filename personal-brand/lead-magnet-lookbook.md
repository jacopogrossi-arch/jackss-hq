## PAGINA 1

### THE LOOKBOOK SYSTEM
*How I build campaign-quality fashion lookbooks with AI — in one afternoon.*

**The rule that makes everything work:**
Every visual stays coherent because the brief was coherent. The AI is not the creative director. You are. The tools only execute decisions you've already made.

---

### STEP 1 — THE CREATIVE BRIEF (60% of the work)

Before opening any tool, lock these six pillars in a written document:

1. **Hero piece** — ONE garment, described exactly: color (use hex codes), materials, hardware, details
2. **World** — ONE setting. Not "outdoor". Something like: "rocky Ligurian coastline, sailboat in background, open sea"
3. **Light** — ONE lighting condition for every shot: time of day, quality, contrast
4. **Palette** — 3-4 colors max, garment + environment together
5. **Model** — one consistent character (a trained AI identity, or a consistent reference set)
6. **Mood** — one sentence. Mine was: "functional, resilient, in motion — he inhabits the garment, he doesn't pose in it"

Then write a shot list: 5 shots, each with frame (full body / three-quarter / close-up / wide), pose, and what the environment does. Real campaigns vary distance and angle inside ONE world — they never change worlds.

---

## PAGINA 2

### STEP 2 — GENERATE (30%)

- One prompt per shot. Every prompt repeats the locked pillars word for word: same garment description, same light, same palette. Repetition is what creates coherence.
- Use a character-consistency feature (e.g. Soul ID on Higgsfield) so the face holds across shots.
- Generate 3-4 variants per shot. Select the strongest. Discard anything that breaks the world — wrong light kills a lookbook faster than a wrong face.
- Include 1 detail shot with no face (fabric, hardware, texture). Real campaigns always do.

### STEP 3 — EDIT (10%)

- Curate down. 5 strong images beat 15 average ones.
- Keep grading consistent: one pass, same treatment on every image.
- Layout matters: white borders, consistent margins, the garment as protagonist.

---

### THE MISTAKES THAT GIVE IT AWAY

- Different light in every image → instant collage effect
- Changing the world between shots → not a campaign, a moodboard
- Over-polished skin and poses → real campaigns have wind, weight, gravity
- Prompting from imagination instead of a written brief → drift
