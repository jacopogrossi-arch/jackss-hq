## 1. Testo dell'esercizio (traccia completa)

La società Alfa commercializza capi di abbigliamento all'ingrosso.

Nella fase di formazione del bilancio al 31.12.n la società Alfa S.r.l. determina un **risultato lordo di competenza dell'esercizio pari a 120.000 euro**. Ai fini della stima delle imposte di competenza risulta che durante l'esercizio "n":

- sono stati rilevati **costi di competenza economica dell'esercizio pari a 30.000 euro** che, in base alla normativa tributaria, risultano **deducibili nell'esercizio successivo "n+1"**;
- sono stati rilevati **costi di competenza economica dell'esercizio, pari a 10.000 euro** che, in base alla normativa tributaria, risultano **indeducibili**;
- sono stati rilevati **ricavi di competenza economica dell'esercizio pari a 20.000 euro** che, per una parte, pari a **4.000 euro, sono imponibili nell'esercizio corrente "n"** e, per la restante parte, pari a **16.000 euro, saranno imponibili negli esercizi successivi**.

Tutto ciò detto e premesso, ipotizzando di trovarsi nella fase di formazione del bilancio dell'esercizio chiuso al 31/12/n, si proceda a:

1. Determinare il reddito d'impresa (imponibile IRES) apportando le opportune riprese fiscali al risultato dell'esercizio, le imposte di competenza dell'esercizio, distinguendo tra quelle correnti, differite e anticipate
2. Effettuare le scritture contabili che rilevano le imposte correnti, differite e anticipate, nonché il debito tributario
3. Indicare nei prospetti contabili di conto economico e di stato patrimoniale di chiusura (bilancio contabile di verifica) le voci che accolgono le imposte correnti, differite e anticipate, nonché il debito tributario
4. Rappresentare negli schemi di bilancio redatto secondo la normativa civilistica gli importi imputati a conto economico e stato patrimoniale con la chiusura dei conti

## 2. Svolgimento — Punto 1: reddito d'impresa e imposte di competenza

### 2.1 Logica delle riprese fiscali

Il procedimento parte dal **risultato lordo civilistico** (120.000) e applica le **riprese fiscali** per arrivare al **reddito imponibile IRES**:

- I **30.000 euro** di costi di competenza economica dell'esercizio ma deducibili solo nell'esercizio n+1 generano una **ripresa in aumento** (perché quest'anno non sono fiscalmente deducibili) → creano una **differenza temporanea deducibile** (si "recupererà" il beneficio fiscale in n+1) → danno origine a **imposte anticipate**.
- I **10.000 euro** di costi indeducibili generano una **ripresa in aumento** definitiva → sono una **differenza permanente** (non si recupera mai).
- I **20.000 euro** di ricavi: solo **4.000** sono imponibili quest'anno (nessuna rettifica, già inclusi nel risultato lordo e tassati subito), mentre i restanti **16.000** sono di competenza economica dell'esercizio ma imponibili fiscalmente solo negli esercizi successivi → generano una **ripresa in diminuzione** di 20.000 (nel senso che vengono tolti dall'imponibile corrente per la parte non ancora tassabile)... 

Nota: dalla tabella del PDF, la ripresa in diminuzione riportata è di **(20.000)** in colonna "Anno n" con contropartita **20.000** in colonna "Diff. Temp. Tassabili". Questo rappresenta l'intero importo dei ricavi (20.000) tolto dal reddito imponibile dell'anno corrente per rinviarne la tassazione, generando una **differenza temporanea tassabile** di 20.000 → dà origine a **imposte differite**.

### 2.2 Tabella di calcolo delle imposte (riportata fedelmente dal PDF)

| Tabella calcolo imposte | Anno "n" | Diff. Temp. Tassabili | Diff. Temp. Deducibili | Diff. Permanente |
|---|---|---|---|---|
| Risultato lordo dell'esercizio | 120.000 | | | |
| Riprese in aumento | 40.000 | | (30.000) | (10.000) |
| Riprese in diminuzione | (20.000) | 20.000 | | |
| **Reddito imponibile** | **140.000** | | | |
| Aliquota IRES (%) | 24% | | | |
| **Imposte correnti** | **33.600** | (140.000 X 24%) | | |
| **Imposte differite** | **4.800** | (20.000 X 24%) | | |
| **Imposte anticipate** | **-7.200** | (-30.000 X 24%) | | |
| **Imposte di competenza del bilancio dell'esercizio** | **31.200** | (= 33.600 + 4.800 − 7.200) | | |

**Come leggere le "Riprese in aumento" = 40.000**: è la somma dei due importi che compaiono nelle colonne "Diff. Temp. Deducibili" (30.000) e "Diff. Permanente" (10.000), cioè 30.000 + 10.000 = 40.000. Questi voci vengono sommate al risultato lordo per ottenere il reddito imponibile.

**Come leggere le "Riprese in diminuzione" = (20.000)**: corrisponde alla differenza temporanea tassabile riportata in colonna (20.000), sottratta dal risultato lordo.

**Calcolo reddito imponibile**: 120.000 + 40.000 − 20.000 = **140.000**

### 2.3 Calcoli intermedi delle imposte

- **Imposte correnti** = 140.000 × 24% = **33.600** (calcolate sul reddito imponibile fiscale effettivo)
- **Imposte differite** = 20.000 × 24% = **4.800** (sulla differenza temporanea tassabile: imposte che si pagheranno in futuro quando i 16.000/20.000 di ricavi diventeranno imponibili — rappresentano un debito potenziale futuro)
- **Imposte anticipate** = -30.000 × 24% = **-7.200** (sulla differenza temporanea deducibile: rappresentano un credito d'imposta futuro, perché i 30.000 di costi non deducibili oggi lo saranno in n+1)
- **Imposte di competenza del bilancio dell'esercizio** = 33.600 + 4.800 − 7.200 = **31.200**

## 3. Svolgimento — Punto 2: scritture contabili

Scritture di rilevazione contabile delle imposte alla chiusura dell'esercizio:

| Dare | Avere | Importo |
|---|---|---|
| Imposte correnti | a Debiti tributari | 33.600 |
| Imposte differite | a Fondo Imposte Differite | 4.800 |
| Imposte Anticipate (credito) | a Imposte anticipate | 7.200 |

**Logica delle scritture:**
- Le **imposte correnti** (33.600) sono un costo di competenza dell'esercizio con contropartita un debito verso l'Erario (Debiti tributari).
- Le **imposte differite** (4.800) sono un costo aggiuntivo di competenza con contropartita un fondo rischi (Fondo Imposte Differite) — rappresentano imposte che si pagheranno effettivamente in futuro.
- Le **imposte anticipate** (7.200) funzionano al contrario: si rileva un **credito** (Imposte Anticipate, voce dell'attivo) con contropartita un **provento/rettifica in diminuzione delle imposte a conto economico** (voce "Imposte anticipate" a conto economico, che riduce il carico fiscale totale).

## 6. Risultato finale

- **Reddito imponibile IRES**: 140.000
- **Imposte correnti**: 33.600 (debito verso Erario)
- **Imposte differite**: 4.800 (fondo rischi, imposte future)
- **Imposte anticipate**: 7.200 (credito d'imposta futuro)
- **Imposte di competenza totali a Conto Economico (voce 20)**: **31.200**
- Voci di bilancio: Fondo Imposte differite (Passivo B) 4.800, Imposte Anticipate (Attivo C) 7.200, Debiti Tributari (Passivo D) 33.600
