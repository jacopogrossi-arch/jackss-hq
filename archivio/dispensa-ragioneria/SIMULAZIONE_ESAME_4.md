### 4.1 — Rimanenze

**Impostazione.** Unità disponibili = 150 + 250 + 400 + 200 = **1.000**; vendute 750 → **rimanenze finali = 250 unità**. Costo complessivo dei beni disponibili = 1.200 + 2.250 + 4.400 + 2.400 = **10.250**.

**a) I tre metodi:**

1. **FIFO** (first in, first out): escono per prime le unità più vecchie → in magazzino restano le **più recenti**: 200 unità dell'acquisto 01/11 a 12,00 (= 2.400) + 50 unità dell'acquisto 01/06 a 11,00 (= 550) → **RF = 2.950**.
2. **LIFO a scatti** (last in, first out): escono per prime le più recenti → restano gli **strati più vecchi**: 150 @ 8,00 (giacenza iniziale) + 100 @ 9,00 (acquisto 01/02) = 1.200 + 900 = **RF = 2.100**.
3. **CMP per periodo**: costo medio = 10.250 / 1.000 = **10,25 €/unità** → RF = 250 × 10,25 = **2.562,50**.

**b) Confronto col mercato (art. 2426, n. 9: minore tra costo e valore di realizzazione desumibile dal mercato).** Valore di mercato = 250 × 10,00 = **2.500**.

| Metodo | Costo | Mercato | Iscrizione in bilancio |
|---|---|---|---|
| FIFO | 2.950 | 2.500 | **2.500** (svalutazione 450) |
| LIFO | 2.100 | 2.500 | **2.100** (il costo è già inferiore) |
| CMP | 2.562,50 | 2.500 | **2.500** (svalutazione 62,50) |

**Risposte multiple:** 1 → **2)** 2.950 · 2 → **3)** 2.100 · 3 → **3) LIFO** (a prezzi crescenti scarica sul costo del venduto i prezzi più recenti e più alti → utile più basso; è l'immagine speculare della domanda delle simulazioni precedenti).

### 4.2 — Imposte

**Qualificazione delle voci:**

1. **Compensi amministratori non pagati (12.000)**: costo in bilancio oggi, deducibile domani (per cassa) → **ripresa in aumento, temporanea deducibile** → **imposte anticipate**.
2. **Svalutazione crediti oltre lo 0,5% (8.000)**: costo in bilancio oggi, deducibile negli esercizi successivi → **ripresa in aumento, temporanea deducibile** → **imposte anticipate**.
3. **Multe e sanzioni (4.000)**: costo mai deducibile → **ripresa in aumento, permanente** → solo più imposte correnti.
4. **Plusvalenza rateizzata (24.000)**: ricavo in bilancio oggi, tassato domani → **ripresa in diminuzione, temporanea tassabile** → **imposte differite**.

**a) Tabella:**

Verifica di coerenza: competenza = correnti + differite − anticipate = 48.000 + 5.760 − 4.800 = **48.960**.

### 4.3 — LIC

**a) Cost-to-cost:** % completamento = costi cumulati / costi totali stimati; ricavi maturati = % × corrispettivo:

Verifica: margine totale = 1.600 + 1.600 + 800 = **4.000** = corrispettivo 20.000 − costi 16.000. ✓

**b)** Anni 1 e 2: in assenza di fatturazione, il valore maturato si iscrive tra le **rimanenze per lavori in corso su ordinazione (C.I.3)** valutate al **corrispettivo maturato** (8.000 a fine anno 1; 16.000 a fine anno 2), con contropartita la **variazione dei lavori in corso su ordinazione (A.3)** del CE. ### 4.4 — OIC 9

**Tabella completata.** Quota di ammortamento attuale = 6.000 / 3 = **2.000/anno**:

| | 2027 | 2028 | 2029 |
|---|---|---|---|
| Margine lordo industriale | 1.800 | 1.600 | 1.400 |
| Ammortamento impianto | −2.000 | −2.000 | −2.000 |
| **Margine industriale netto** | **−200** | **−400** | **−600** |
| Capacità di ammortamento (2027) | **1.800** | | |

**Ragionamento (approccio semplificato OIC 9):**

1. La **capacità di ammortamento** è il margine disponibile *prima* dell'ammortamento: è quanto la gestione riesce a "coprire" del costo del cespite.
2. **Valore d'uso** ≈ somma delle capacità di ammortamento residue = 1.800 + 1.600 + 1.400 = **4.800**.
3. **Valore recuperabile** = maggiore tra prezzo netto di vendita (4.200) e valore d'uso (4.800) = **4.800**.
4. Confronto: valore contabile 6.000 > recuperabile 4.800 → **svalutazione di 1.200** → l'impianto è iscritto a **4.800**.
5. Nuova quota di ammortamento = 4.800 / 3 = **1.600/anno**.

**Scrittura contabile:**

```
Svalutazione immobilizzazioni (B.10.c)  a  Impianti (o Fondo svalutazione)  1.200
```

**Risposte multiple:** 1 → **2)** 4.800 · 2 → **3)** 1.600 · 3 → **1)** 1.200 (se PNV = 5.500 > valore d'uso 4.800, il recuperabile diventa 5.500: svalutazione = 6.000 − 5.500 = **500** → risposta corretta **2)** 500. Attenzione al distrattore: la svalutazione cambia perché cambia il *maggiore* tra i due valori).

### 4.5 — Indici

1. **Tesoreria secondario** = (liquidità immediate + differite) / passività correnti = (100+180)/300 = **0,93** → leggermente sotto l'unità: le passività a breve non sono interamente coperte senza contare il magazzino; situazione da monitorare ma non allarmante.
2. **Disponibilità (current ratio)** = attivo circolante / passività correnti = 400/300 = **1,33** → soddisfacente (soglia di attenzione: < 1).
3. **Struttura primario** = PN / attivo fisso = 400/600 = **0,67** → il patrimonio netto da solo non copre l'attivo fisso (valore < 1 fisiologico se il secondario è a posto).
4. **Struttura secondario** = (PN + passività consolidate) / attivo fisso = 700/600 = **1,17** → l'attivo fisso è interamente finanziato da fonti durevoli: equilibrio strutturale rispettato.
5. **ROI** = reddito operativo / capitale investito = 150/1.000 = **15%**.
6. **ROS** = reddito operativo / ricavi = 150/1.500 = **10%**.
7. **ROE** = utile netto / PN = 72/400 = **18%** → superiore al ROI: la leva finanziaria lavora a favore (il costo del debito è inferiore al rendimento operativo).
