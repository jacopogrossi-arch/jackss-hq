## Soluzioni Simulazione 1

### 1.1 — Rimanenze

**Impostazione.** Unità disponibili = 100 + 200 + 300 + 200 = **800**; vendute 600 → **rimanenze finali = 200 unità**. Costo complessivo dei beni disponibili = 1.000 + 2.400 + 3.900 + 3.000 = **10.300**.

**a) I tre metodi:**

1. **FIFO** (first in, first out): escono per prime le unità più vecchie → in magazzino restano le **più recenti**: 200 unità dell'acquisto 01/10 a 15,00 → **RF = 3.000**.
2. **LIFO a scatti** (last in, first out): escono per prime le più recenti → restano gli **strati più vecchi**: 100 @ 10,00 (giacenza iniziale) + 100 @ 12,00 (acquisto 01/03) = 1.000 + 1.200 = **RF = 2.200**.
3. **CMP per periodo**: costo medio = 10.300 / 800 = **12,875 €/unità** → RF = 200 × 12,875 = **2.575**.

**b) Confronto col mercato (art. 2426, n. 9: minore tra costo e valore di realizzazione desumibile dal mercato).** Valore di mercato = 200 × 12,00 = **2.400**.

| Metodo | Costo | Mercato | Iscrizione in bilancio |
|---|---|---|---|
| FIFO | 3.000 | 2.400 | **2.400** (svalutazione 600) |
| LIFO | 2.200 | 2.400 | **2.200** (il costo è già inferiore) |
| CMP | 2.575 | 2.400 | **2.400** (svalutazione 175) |

**Risposte multiple:** 1 → **2)** 3.000 · 2 → **1)** 2.200 · 3 → **3) FIFO** (a prezzi crescenti lascia in magazzino i costi più alti → costo del venduto più basso → utile più alto).

### 1.2 — Imposte

**Logica delle riprese.** Si parte dal risultato civilistico e si applicano le variazioni fiscali:

1. **Compensi amministratori non pagati (20.000)**: costo in bilancio oggi, deducibile domani → **ripresa in aumento, differenza temporanea deducibile** → genera **imposte anticipate**.
2. **Sanzioni (5.000)**: costo mai deducibile → **ripresa in aumento, differenza permanente** → solo più imposte correnti, nessuna fiscalità differita.
3. **Plusvalenza rateizzata (20.000)**: ricavo in bilancio oggi, tassato domani → **ripresa in diminuzione, differenza temporanea tassabile** → genera **imposte differite**.

**a) Tabella:**

| | Anno "n" | Temp. tassabili | Temp. deducibili | Permanenti |
|---|---|---|---|---|
| Risultato lordo | 150.000 | | | |
| Riprese in aumento | +25.000 | | 20.000 | 5.000 |
| Riprese in diminuzione | −20.000 | 20.000 | | |
| **Reddito imponibile** | **155.000** | | | |
| Imposte correnti (24%) | **37.200** | | | |
| Imposte differite (20.000 × 24%) | **4.800** | | | |
| Imposte anticipate (20.000 × 24%) | **−4.800** | | | |
| **Imposte di competenza** | **37.200** | | | |

Verifica di coerenza: competenza = correnti + differite − anticipate = 37.200 + 4.800 − 4.800 = 37.200. (Le due differenze temporanee di pari importo si compensano: il carico di competenza coincide per caso con le correnti.)

### 1.3 — OIC 9

**Tabella completata.** Quota di ammortamento attuale = 4.000 / 4 = **1.000/anno**:

| | 2027 | 2028 | 2029 | 2030 |
|---|---|---|---|---|
| Margine lordo industriale | 900 | 800 | 700 | 600 |
| Ammortamento impianto | −1.000 | −1.000 | −1.000 | −1.000 |
| **Margine industriale netto** | **−100** | **−200** | **−300** | **−400** |
| Capacità di ammortamento (2027) | **900** | | | |

**Ragionamento (approccio semplificato OIC 9):**

1. La **capacità di ammortamento** è il margine disponibile *prima* dell'ammortamento: è quanto la gestione riesce a "coprire" del costo del cespite.
2. **Valore d'uso** ≈ somma delle capacità di ammortamento residue = 900 + 800 + 700 + 600 = **3.000**.
3. **Valore recuperabile** = maggiore tra prezzo netto di vendita (2.800) e valore d'uso (3.000) = **3.000**.
4. Confronto: valore contabile 4.000 > recuperabile 3.000 → **svalutazione di 1.000** → l'impianto è iscritto a **3.000**.
5. Nuova quota di ammortamento = 3.000 / 4 = **750/anno**.

**Risposte multiple:** 1 → **2)** 3.000 · 2 → **3)** 750 · 3 → **2)** 3.500 (se PNV = 3.500 > valore d'uso 3.000, il recuperabile diventa 3.500: svalutazione di 500 e nuovo valore da ammortizzare 3.500).

### 1.4 — Indici

1. **Tesoreria secondario** = (liquidità immediate + differite) / passività correnti = (50+150)/200 = **1,00** → le passività a breve sono coperte esattamente dalle liquidità, senza contare il magazzino: equilibrio al limite ma accettabile.
2. **Disponibilità (current ratio)** = attivo circolante / passività correnti = 300/200 = **1,50** → soddisfacente (soglia di attenzione: < 1).
3. **Struttura primario** = PN / attivo fisso = 350/500 = **0,70** → il patrimonio netto da solo non copre l'attivo fisso (valore < 1 fisiologico se il secondario è a posto).
4. **Struttura secondario** = (PN + passività consolidate) / attivo fisso = 600/500 = **1,20** → l'attivo fisso è interamente finanziato da fonti durevoli: equilibrio strutturale rispettato.
5. **ROI** = reddito operativo / capitale investito = 120/800 = **15%**.
6. **ROS** = reddito operativo / ricavi = 120/1.000 = **12%**.
7. **ROE** = utile netto / PN = 60/350 = **17,1%** → superiore al ROI: la leva finanziaria lavora a favore (il costo del debito è inferiore al rendimento operativo).

### Domande lampo — Sim. 1

1. **Stato patrimoniale, conto economico, rendiconto finanziario, nota integrativa** (art. 2423, co. 1). Abbreviato: esonero dal rendiconto (e possibile esonero dalla relazione sulla gestione, art. 2435-bis, co. 7). Micro: esonero anche dalla nota integrativa (con informazioni in calce allo SP).
2. **Utili sperati: mai** (si rilevano solo se realizzati alla chiusura); **perdite presunte: sempre**, anche se conosciute dopo la chiusura.
3. Verifica che l'impresa costituisca **un complesso funzionante destinato alla produzione di reddito** su un orizzonte di **almeno 12 mesi** dalla data di riferimento del bilancio.
4. **5% dell'utile netto annuo** fino a raggiungere **il quinto (20%) del capitale sociale** (art. 2430).
