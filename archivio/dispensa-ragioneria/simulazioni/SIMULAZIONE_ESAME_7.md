### 7.1 — Rimanenze

**Impostazione.** Unità disponibili = 300 + 350 + 250 + 100 = **1.000**; vendute 750 → **rimanenze finali = 250 unità**. Costo complessivo dei beni disponibili = 900 + 1.400 + 1.250 + 650 = **4.200**.

**a) I tre metodi:**

1. **FIFO**: restano le **più recenti**: 100 unità dell'acquisto 01/11 a 6,50 (= 650) + 150 unità dell'acquisto 01/08 a 5,00 (= 750) → **RF = 1.400**.
2. **LIFO a scatti**: le RF (250) sono **inferiori alla giacenza iniziale** (300) → si intacca lo strato più vecchio: 250 @ 3,00 → **RF = 750**.
3. **CMP per periodo**: costo medio = 4.200 / 1.000 = **4,20 €/unità** → RF = 250 × 4,20 = **1.050**.

**b) Confronto col mercato (art. 2426, n. 9).** Valore di mercato = 250 × 4,00 = **1.000**.

| Metodo | Costo | Mercato | Iscrizione in bilancio |
|---|---|---|---|
| FIFO | 1.400 | 1.000 | **1.000** (svalutazione 400) |
| LIFO | 750 | 1.000 | **750** (il costo è già inferiore) |
| CMP | 1.050 | 1.000 | **1.000** (svalutazione 50) |

**Risposte multiple:** 1 → **2)** 750 · 2 → **3)** 1.000 · 3 → **3) LIFO** (a prezzi crescenti scarica a CE i costi più recenti, cioè più alti → costo del venduto maggiore → utile minore; è l'immagine speculare della domanda della sim. 6).

### 7.2 — Imposte

**Qualificazione delle voci:**

1. **Compensi amministratori non pagati (20.000)**: costo oggi, deducibile per cassa domani → **ripresa in aumento, temporanea deducibile** → **anticipate**.
2. **Svalutazione crediti oltre lo 0,5% (8.000)**: costo oggi, deducibile in futuro → **ripresa in aumento, temporanea deducibile** → **anticipate**.
3. **Sanzioni e multe (4.000)**: mai deducibili → **ripresa in aumento, permanente**.
4. **Plusvalenza rateizzata (10.000)**: tutta nell'utile civilistico oggi, ma tassata in 5 quote da 2.000 (n … n+4) → nell'anno "n" si **riprende in diminuzione 8.000** (i 4/5 che saranno tassati in futuro) → **temporanea tassabile** → **differite**.

**a) Tabella:**

| | Anno "n" | Temp. tassabili | Temp. deducibili | Permanenti |
|---|---|---|---|---|
| Risultato lordo | 250.000 | | | |
| Riprese in aumento | +32.000 | | 28.000 | 4.000 |
| Riprese in diminuzione | −8.000 | 8.000 | | |
| **Reddito imponibile** | **274.000** | | | |
| Imposte correnti (24%) | **65.760** | | | |
| Imposte differite (8.000 × 24%) | **1.920** | | | |
| Imposte anticipate (28.000 × 24%) | **−6.720** | | | |
| **Imposte di competenza** | **60.960** | | | |

Verifica di coerenza: competenza = correnti + differite − anticipate = 65.760 + 1.920 − 6.720 = **60.960**.

**b) Scritture (31/12/n):**

```
Imposte correnti                a   Debiti tributari                 65.760
Imposte differite               a   Fondo imposte differite           1.920
Attività per imposte anticipate a   Imposte anticipate (CE)           6.720
```

**c)** Voce 20 CE = 65.760 + 1.920 − 6.720 = **60.960**.

**Risposte multiple:** 1 → **3)** 6.720 (= (20.000 + 8.000) × 24%) · 2 → **2)** temporanea **tassabile**: utile civilistico oggi, tassazione futura → imposte differite.

### 7.4 — Rendiconto finanziario

**a) Schema OIC 10 (metodo indiretto):**

**A — Gestione reddituale:**

| | € |
|---|---|
| Utile dell'esercizio | 500 |
| + Ammortamenti (costo non monetario) | +300 |
| − Plusvalenza da cessione cespite | −100 |
| − Aumento crediti commerciali | −150 |
| + Diminuzione rimanenze | +80 |
| + Aumento debiti v/fornitori | +120 |
| **Flusso A** | **+750** |

**B — Attività di investimento:**

| | € |
|---|---|
| Prezzo di realizzo cessione macchinario | +400 |
| Acquisto impianto | −600 |
| **Flusso B** | **−200** |

**C — Attività di finanziamento:**

| | € |
|---|---|
| Rimborso quota mutuo | −200 |
| Dividendi pagati | −150 |
| **Flusso C** | **−350** |

**Variazione disponibilità liquide** = 750 − 200 − 350 = **+200** → **cassa finale = 250 + 200 = 450**.

**b) Perché si sottrae la plusvalenza.** L'**intero prezzo incassato (400)** è già esposto nell'attività di investimento. La plusvalenza (100) è dentro l'utile da cui parte la sezione A: se non la si sottraesse, quei 100 verrebbero **contati due volte** (una in A dentro l'utile, una in B dentro i 400). Inoltre il flusso da cessione di cespiti non è gestione reddituale ma investimento: la rettifica riporta ogni flusso nella sezione di pertinenza.

**Risposte multiple:** 1 → **2)** 750 · 2 → **2)** riduce il flusso (ricavi contabilizzati ma non ancora incassati) · 3 → **3)** attività di finanziamento (remunerazione del capitale proprio).

### 7.5 — Indici

1. **Tesoreria secondario** = (liquidità immediate + differite) / passività correnti = (60+180)/220 = **1,09** → sopra 1: le liquidità coprono da sole le passività a breve, senza dover contare sul magazzino.
2. **Disponibilità (current ratio)** = attivo circolante / passività correnti = 360/220 = **1,64** → buon margine di sicurezza.
3. **Struttura primario** = PN / attivo fisso = 380/500 = **0,76** → il PN da solo non copre l'attivo fisso (fisiologico).
4. **Struttura secondario** = (PN + passività consolidate) / attivo fisso = 640/500 = **1,28** → le fonti durevoli coprono l'attivo fisso con un margine che finanzia anche parte del circolante: equilibrio strutturale solido.
5. **ROI** = reddito operativo / capitale investito = 132/860 = **15,3%**.
6. **ROS** = reddito operativo / ricavi = 132/1.200 = **11%**.
7. **ROE** = utile netto / PN = 66/380 = **17,4%** → **superiore al ROI**: la leva finanziaria lavora **a favore** (il costo del debito è inferiore al rendimento operativo). Quadro complessivo opposto a quello della sim. 6: liquidità adeguata, struttura equilibrata, leva favorevole.

### Domande lampo — Sim. 7

1. **Gestione reddituale (A), attività di investimento (B), attività di finanziamento (C)**; la grandezza di riferimento sono le **disponibilità liquide** (cassa e banche attive, al netto degli scoperti se assimilati).
2. Solo l'**avviamento derivativo** (acquisito a titolo oneroso, es. con una cessione d'azienda) può essere iscritto: **con il consenso del collegio sindacale**, nei limiti del **costo sostenuto**, e va ammortizzato secondo la sua vita utile (se non stimabile attendibilmente, entro 10 anni). L'avviamento **originario** (autogenerato) non è mai iscrivibile.
3. Perché gli ammortamenti sono un **costo non monetario**: riducono l'utile ma non generano alcuna uscita di cassa nell'esercizio. Partendo dall'utile, vanno risommati per risalire al flusso di cassa effettivo.
4. Il **5% dell'utile netto annuale**, finché la riserva non ha raggiunto **il quinto (20%) del capitale sociale**; se scende sotto (es. dopo un aumento di CS), l'obbligo riprende.
