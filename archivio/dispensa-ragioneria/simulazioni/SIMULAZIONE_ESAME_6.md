### 6.1 — Rimanenze

**Impostazione.** Unità disponibili = 250 + 350 + 300 + 100 = **1.000**; vendute 800 → **rimanenze finali = 200 unità**. Costo complessivo dei beni disponibili = 1.000 + 1.750 + 1.800 + 700 = **5.250**.

**a) I tre metodi:**

1. **FIFO**: restano le **più recenti**: 100 unità dell'acquisto 01/12 a 7,00 (= 700) + 100 unità dell'acquisto 01/09 a 6,00 (= 600) → **RF = 1.300**.
2. **LIFO a scatti**: restano gli **strati più vecchi**: 200 @ 4,00 (giacenza iniziale) → **RF = 800**.
3. **CMP per periodo**: costo medio = 5.250 / 1.000 = **5,25 €/unità** → RF = 200 × 5,25 = **1.050**.

**b) Confronto col mercato (art. 2426, n. 9).** Valore di mercato = 200 × 5,00 = **1.000**.

| Metodo | Costo | Mercato | Iscrizione in bilancio |
|---|---|---|---|
| FIFO | 1.300 | 1.000 | **1.000** (svalutazione 300) |
| LIFO | 800 | 1.000 | **800** (il costo è già inferiore) |
| CMP | 1.050 | 1.000 | **1.000** (svalutazione 50) |

**Risposte multiple:** 1 → **3)** 800 · 2 → **2)** 1.000 · 3 → **3) FIFO** (a prezzi crescenti lascia in magazzino gli acquisti più recenti, cioè i costi più alti; per lo stesso motivo espone anche l'utile più alto).

### 6.2 — Imposte

**Qualificazione delle voci:**

1. **Compensi amministratori non pagati (15.000)**: costo oggi, deducibile domani per cassa → **ripresa in aumento, temporanea deducibile** → **anticipate**.
2. **Manutenzioni oltre plafond (5.000)**: costo oggi, deducibile nei 5 successivi → **ripresa in aumento, temporanea deducibile** → **anticipate**.
3. **Costi auto indeducibili (7.000)**: mai deducibili → **ripresa in aumento, permanente**.
4. **Interessi attivi di mora non incassati (6.000)**: ricavo in bilancio oggi, tassato quando incassato → **ripresa in diminuzione, temporanea tassabile** → **differite**.

**a) Tabella:**

|---|---|---|---|---|
| Risultato lordo | 180.000 | | | |
| Riprese in aumento | +27.000 | | 20.000 | 7.000 |
| Riprese in diminuzione | −6.000 | 6.000 | | |
| **Reddito imponibile** | **201.000** | | | |
| Imposte correnti (24%) | **48.240** | | | |

| Imposte anticipate (20.000 × 24%) | **−4.800** | | | |
| **Imposte di competenza** | **44.880** | | | |

Verifica di coerenza: competenza = correnti + differite − anticipate = 48.240 + 1.440 − 4.800 = **44.880**.

**b) Scritture (31/12/n):**


**Risposte multiple:** 1 → **2)** 4.800 (= (15.000 + 5.000) × 24%) · 2 → **3)** temporanea **tassabile**: il ricavo è in bilancio oggi ma sarà tassato in futuro → genera imposte differite.

### 6.3 — LIC

**a) Cost-to-cost:** % completamento = costi cumulati / costi totali stimati; ricavi maturati = % × corrispettivo:

| | Anno 1 | Anno 2 | Anno 3 |
|---|---|---|---|
| % completamento | 3.600/12.000 = **30%** | 9.600/12.000 = **80%** | **100%** |
| Ricavi maturati cumulati | **4.500** | **12.000** | **15.000** |
| Ricavi di competenza | **4.500** | 12.000−4.500 = **7.500** | **3.000** |
| Margine dell'anno | 4.500−3.600 = **900** | 7.500−6.000 = **1.500** | 3.000−2.400 = **600** |

Verifica: margine totale = 900 + 1.500 + 600 = **3.000** = corrispettivo 15.000 − costi 12.000. ✓

**b)** Anni 1 e 2: in assenza di fatturazione, il valore maturato si iscrive tra le **rimanenze per lavori in corso su ordinazione (C.I.3)** valutate al **corrispettivo maturato** (4.500 a fine anno 1; 12.000 a fine anno 2), con contropartita la **variazione dei lavori in corso su ordinazione (A.3)** del CE. All'anno 3, alla consegna, si rileva il **ricavo di vendita (A.1) per 15.000** e si stornano integralmente le rimanenze.

**Risposte multiple:** 1 → **3)** 3.000 (= 15.000 − 12.000: il metodo cambia la *distribuzione* del margine negli anni, mai il suo totale) · 2 → **2)** interamente all'anno 3, alla consegna (le rimanenze nel frattempo si valutano al costo).

### 6.4 — OIC 9

**Tabella completata.** Quota di ammortamento attuale = 8.000 / 4 = **2.000/anno**:

| | 2027 | 2028 | 2029 | 2030 |
|---|---|---|---|---|
| Margine lordo industriale | 1.900 | 1.700 | 1.500 | 1.300 |
| Ammortamento impianto | −2.000 | −2.000 | −2.000 | −2.000 |
| **Margine industriale netto** | **−100** | **−300** | **−500** | **−700** |
| Capacità di ammortamento (2027) | **1.900** | | | |

**Ragionamento (approccio semplificato OIC 9):**

1. **Valore d'uso** ≈ somma delle capacità di ammortamento residue = 1.900 + 1.700 + 1.500 + 1.300 = **6.400**.
2. **Valore recuperabile** = maggiore tra prezzo netto di vendita (5.600) e valore d'uso (6.400) = **6.400**.
3. Confronto: valore contabile 8.000 > recuperabile 6.400 → **svalutazione di 1.600** → l'impianto è iscritto a **6.400**.
4. Nuova quota di ammortamento = 6.400 / 4 = **1.600/anno**.

**Scrittura contabile:**

```
Svalutazione immobilizzazioni (B.10.c)  a  Impianti (o Fondo svalutazione)  1.600
```

**Risposte multiple:** 1 → **2)** 6.400 · 2 → **3)** 1.600 · 3 → **2)** 1.000 (se PNV = 7.000 > valore d'uso 6.400, il recuperabile diventa 7.000: svalutazione = 8.000 − 7.000 = 1.000).

### 6.5 — Indici

1. **Tesoreria secondario** = (liquidità immediate + differite) / passività correnti = (40+120)/250 = **0,64** → **criticità**: le liquidità coprono solo due terzi delle passività a breve; l'azienda dipende dallo smobilizzo del magazzino per far fronte agli impegni correnti.
2. **Disponibilità (current ratio)** = attivo circolante / passività correnti = 250/250 = **1,00** → esattamente al limite della soglia di attenzione: nessun margine di sicurezza.
3. **Struttura primario** = PN / attivo fisso = 280/450 = **0,62** → il patrimonio netto copre poco più della metà dell'attivo fisso.
4. **Struttura secondario** = (PN + passività consolidate) / attivo fisso = 450/450 = **1,00** → le fonti durevoli coprono esattamente l'attivo fisso, senza alcun margine per finanziare il circolante: equilibrio strutturale al limite.
5. **ROI** = reddito operativo / capitale investito = 90/700 = **12,9%**.
6. **ROS** = reddito operativo / ricavi = 90/900 = **10%**.
7. **ROE** = utile netto / PN = 28/280 = **10%** → **inferiore al ROI**: la leva finanziaria lavora *contro* (il costo dell'indebitamento erode la redditività operativa). Quadro complessivo: redditività operativa discreta ma struttura finanziaria tesa, con liquidità insufficiente e leva sfavorevole.
