## Nuovi Modelli

**Claude Sonnet 4.6** — Upgrade completo in coding, computer use, ragionamento su contesti lunghi, finestra di contesto da 1M token (beta).

**Claude Mythos Preview** — Modello più avanzato di Anthropic, eccezionale in cybersecurity. Non disponibile al pubblico; accesso riservato a 11 aziende nel Project Glasswing.

## Claude Code (aggiornamento aprile 2026)

- Redesign desktop: terminale integrato, side chat, diff viewer migliorato, pannelli riorganizzabili
- Routines: automazioni programmabili su infrastruttura cloud (Mac non deve essere acceso)
- Prompt caching: controlli a 5 min forzato e sessioni da 1 ora
- Session recap: riepilogo automatico delle sessioni

## Claude Cowork

GA su macOS e Windows nell'app Claude Desktop. Analytics espansi, OpenTelemetry, role-based access controls (Enterprise).

## API e Piattaforma

- ant CLI: client da riga di comando per API Claude, integrazione nativa con Claude Code, versioning YAML
- Messages API su Amazon Bedrock: research preview
- Advisor tool (beta pubblica): modello esecutore veloce + modello advisor ad alta intelligenza
